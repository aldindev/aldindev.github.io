Pinoy Gig Market

Pinoy Gig Market is a web application designed to connect Filipino freelancers with clients seeking their services.

Key Features:

  • User Roles: Users can register as freelancers, clients, or both.
  • Gig Posting & Acceptance: Clients can post gigs and freelancers can apply for them.
  • Profile Management: Build and manage your freelancer or client profile.
  • Communication Tools: Connect with other users, send messages, and collaborate on projects.

Technical Stack:

  • ASP.NET MVC (Backend Framework)
  • Razor, Bootstrap (Frontend Framework)
  • SQL Database (Data Storage)
  • IIS (hosted on personal home Web Server)

This is a functional and ongoing project with continuous improvements planned.
